What is MUCK?

MUCK is software for making sense out of massive, messy collections of documents. This toolkit can be used to take directories of unwieldy amounts of texts, extract useful information (both metadata and linguistic features like parts-of-speech, named entities, and parsed dependencies), and construct a searchable document database with all extracted information readily available.

History

MUCK was initially written to handle large collections of news content and government data, such as news archives, Congressional press releases/floor speeches, and legislation. The toolkit was designed to answer many different questions that required natural language understanding of these texts.

Tech

At the most basic level, MUCK is a standalone web service around the Stanford NLP group's JavaNLP library. It can run on your own machine as a Tomcat servlet that provides a simple RESTful API to the NLP library.

MUCK was also designed for use on AWS platforms, and can scale up by distributing across EC2 instances. This means that you can add and remove nodes depending on the scale of a text collection.

MUCK is also intended to complement Amazon Mechanical Turk, primarily for creating training data needed in classification tasks (topics, sentiment, etc).
